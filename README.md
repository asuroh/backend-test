# Backend-test

## Requirement
- install laravel
- install mysql 
- install postgresql
- install postman

## Documentasi
Step 1 
clone repository

Step 2
- create file .env copy dari file .env.example
- configuration env

Step 3
```
- php artisan migrate
```

Step 3
```
- php artisan serve
```

Step 5
- Configuration Postman
Postman collection : 
    in file backend-test.postman_collection.json
Postman Env : 
    in file Backend-test.postman_environment.json
